Simple Bitbucket API wrapper for PHP >= 5.3.2.

## Requirements

* PHP >= 5.3 with [cURL](http://php.net/manual/en/book.curl.php) extension.
* [Buzz](https://github.com/kriswallsmith/Buzz) library,
* PHPUnit to run tests. ( _optional_ )
